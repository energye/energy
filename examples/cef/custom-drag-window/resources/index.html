说明

示例使用2种试实现通过html元素拖拽窗口, 区别于LCL和VF窗口

通过设置CSS属性 "--webkit-app-region: drag" 或 "-webkit-app-region: drag"

1. "--webkit-app-region: drag" 通过JS扩展实现, LCL 支持 Windows, MacOS

2 "-webkit-app-region: drag" 通过窗口消息实现, VF 支持 Windows, MacOS, Linux

OS推荐-根据不同的窗口

  Windows: -webkit-app-region

  MacOS: --webkit-app-region

  Linux: -webkit-app-region

不选择文本: -webkit-user-select: none;
